# Robomow ESP Web Flasher

Pages website for installing Robomow ESP32-S3 N16R8 firmware from a browser using ESP Web Tools.

## Features

* Browser-based firmware installation via Web Serial.
* Russian and English UI, with Russian as the default language.
* Light and dark themes.
* Firmware profiles for ESP32-S3 N16R8.
* ESP Web Tools manifests for merged firmware binaries.
