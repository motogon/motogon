Установка прошивки

Выберите профиль и запустите установку. UART‑логи в этом интерфейсе не выводятся.

Браузер не поддерживает Web Serial. Используйте Chrome, Edge или другой Chromium-браузер на ПК. Web Serial работает только на HTTPS или localhost. Откройте страницу через GitHub Pages.

Перед прошивкой отключите питание моторов/ножа. Для первой установки обычно выбирайте стирание flash.

Порядок действий

  1. Подключите ESP32/ESP32-S3 к компьютеру качественным USB‑кабелем с передачей данных.
  2. Откройте эту страницу в Chrome или Edge. На iOS Web Serial не работает.
  3. Нажмите «Подключить и прошить», выберите USB‑порт и подтвердите запись.
  4. После завершения дождитесь перезагрузки ESP и проверьте точку доступа или веб‑интерфейс прошивки.

Безопасность

Для Robomow это сервисная прошивка. Перед установкой физически исключите запуск ножа и движения: снимите питание силовой части, не держите косилку на земле в активном состоянии, используйте аварийную остановку.

Файлы прошивки

Положите merged binary в каталог firmware и обновите manifest. Для ESP32 с ESP-IDF/Arduino обычно нужен один объединённый bin с offset 0. 

firmware/
  manifest.json
  manifest-esp32.json
  manifest-esp32s3.json
  robomow-esp32-merged.bin
  robomow-esp32s3-merged.bin

Публикация

Сайт готов для GitHub Pages. После push включите Pages из GitHub Actions или используйте приложенный workflow. 

git init
git add .
git commit -m "Initial Robomow web flasher"
git branch -M main
git remote add origin git@github.com:OWNER/REPO.git
git push -u origin main

UART‑логи отключены в интерфейсе сайта

На странице нет режима просмотра UART. Manifest настроен без Improv‑ожидания, чтобы установщик не переходил в сервисный режим просмотра устройства после прошивки. Это не блокирует физический UART для человека с доступом к плате и внешним терминалом.
