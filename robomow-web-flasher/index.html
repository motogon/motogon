Firmware installation

Target device: Robomow ESP32-S3.

Firmware profile Robomow ESP32-S3
This browser does not support Web Serial. Use Chrome, Edge, or another Chromium browser on a desktop computer. Web Serial works only over HTTPS or localhost. Open this page through GitHub Pages.

Before flashing, disconnect blade and motor power. For first installation, erase flash when prompted.

Steps

  1. Connect the ESP32-S3 to your computer using a good USB data cable.
  2. Open this page in Chrome or Edge. Web Serial does not work on iOS.
  3. Click “Connect and install”, select the USB port, and confirm flashing.
  4. After flashing, wait for the ESP to reboot and open the firmware web interface at http://192.168.4.1/.
http://192.168.4.1/

Safety

This is service firmware for Robomow. Before installation, physically prevent blade start and mower movement: disconnect power from the drive/blade circuits, keep the mower in a safe service position, and use emergency stop when needed.
