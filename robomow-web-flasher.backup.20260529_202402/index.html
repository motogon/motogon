ESP32-S3 N16R8 Web Serial GitHub Pages

Установка прошивки

Выберите профиль и запустите установку. UART‑логи в этом интерфейсе не выводятся.

Браузер не поддерживает Web Serial. Используйте Chrome, Edge или другой Chromium-браузер на ПК. Web Serial работает только на HTTPS или localhost. Откройте страницу через GitHub Pages.

Перед прошивкой отключите питание моторов/ножа. Для первой установки обычно выбирайте стирание flash.
